About

I am a recent Ohio State graduate with a Bachelor's of Science and Computer and Information Sciences. I have a specialization in Computer Systems and have recently also taken a major interest in web development. I have experience working in Data Science and backend systems as well as front-end mobile application development. I am currently open to work and looking for an internship or entry level position in software engineering!

In my most recent project I worked with medical professionals at the Wexner Medical Center to create an iOS/Android application to support physicians in perioperative assessments. Our application used the React Native framework to build out our project for both iOS and Android platforms at once. Our application broke down the various assessments to be performed by organ system after which the user can select and perform the assessment. Throughout this project I have gained relevant experience working with clients and demoing various UI approaches as well as working on backend algorithm design to implement our assessments in components. In addition, working with a team of 5 on this project we were able to implement scrum methodology. We used Trello to host our backlog and broke our project up into sprints for each of the major components.
